# Run Feb Code Evals

## Get Started

NOTE: This is MacOS only for now!

Clone the Feb Code repo:

```sh
git clone https://github.com/FebVetGit/Feb-Code.git
cd Feb-Code
```

Run the setup script:

```sh
cd evals
./scripts/setup.sh
```

Navigate to [localhost:3000](http://localhost:3000/) in your browser.
